## Resume Review

Provide a comprehensive review with rewrites for maximum impact.

### Phase 1 — First Impression (30-second scan)

Evaluate what a recruiter sees in 30 seconds:
- Is the current/recent role clear and compelling?
- Are there impact numbers visible in the first scan?
- Is the tech stack visible and relevant to the target role?
- Is the format clean and ATS-parseable?

Score: First impression (1-5) with specific reason.

### Phase 2 — Bullet Point Analysis

Review every bullet point:

For each weak bullet, provide a rewrite:
```
BEFORE: [original text]
AFTER:  [improved version with action verb + result + metric]
```

Check:
- Does it start with a strong action verb?
- Does it quantify impact (latency, users, cost, time saved)?
- Is it specific enough to be credible?
- Does it show ownership or just participation?

### Phase 3 — Skills Section

Review skills relevance:
- Are skills ordered correctly (most relevant to target role first)?
- Are there outdated/irrelevant skills that dilute the section?
- Are there missing skills that should be included?
- Is the grouping by category clear?

### Phase 4 — ATS Analysis

Identify keyword gaps:
- Compare against a target job description if provided
- List critical keywords from JD that are missing from resume
- Suggest where to naturally incorporate them

### Phase 5 — Level and Scope Assessment

Does the resume reflect the target level?
- L4/Mid: shows ownership of features, not just tasks
- L5/Senior: shows architectural decisions, cross-team impact, mentoring
- L6+/Staff: shows org-wide influence, platform thinking, technical strategy

Flag scope mismatches between actual experience and how it's presented.

### Output

Provide:
1. **Scores** — First impression, Impact, Relevance, Clarity (1-5 each)
2. **Top 3 priorities** — the changes with highest ROI
3. **All bullet rewrites** — before/after for every weak bullet
4. **Summary statement** — a stronger 2-3 line opening summary
5. **ATS keyword list** — keywords to add and where
6. **Final assessment** — is this resume competitive for the target role?
